# The Deliberation Gradient

## Mechanistic Signatures of Dual-Process Cognition in LLMs

**HUSAI Research Project -- Spring/Summer 2026**

---

### What is this project?

We are investigating whether LLMs exhibit mechanistically distinct processing modes analogous to human "fast" (heuristic) and "slow" (deliberative) cognition, using five complementary interpretability methods on cognitive bias benchmarks. The headline experiment compares Llama-3.1-8B-Instruct against DeepSeek-R1-Distill-Llama-8B -- same architecture, different training -- to isolate what reasoning distillation actually changes inside a model.

### Why does it matter?

Understanding when models genuinely deliberate versus pattern-match is a core AI safety question: it determines when we can trust model outputs on novel problems and when oversight is most needed. This project produces mechanistic evidence -- not behavioral observations -- about how reasoning training changes model internals, contributing to both the interpretability and alignment research agendas.

### What you'll learn

- **Mechanistic interpretability**: linear probing, sparse autoencoders (SAEs), activation analysis, causal interventions
- **Research-grade statistics**: permutation testing, BH-FDR correction, bootstrap confidence intervals, effect sizes
- **Engineering**: PyTorch hooks, HDF5 data pipelines, HPC/GPU workflows, Hydra config management
- **Scientific practice**: pre-registration, falsification-first methodology, reproducibility standards
- **Academic writing**: paper drafting, figure design, conference submission process
